# mittoballe
This repository contains the TEI encoding for my edition of the reel-to-reel audio tape preserving the recording of Costantina Zotti Tanti reading her manuscript in the Cimbrian language of Sette Comuni (Vicenza, Italia).


## Important note
The XML file contained in this repository was first created in 2016 and it is currently (2026) being updated in the context of a project for a Cimbrian corpus. The interface for the digital edition of this material will therefore take place in the mentioned project. More on this coming soon! The commentary to the text, which is available in German and Italian, can be found here: Cugliana, Elisa. 2015. „Mittobàlle: Eine TEI-konforme digitale Ausgabe“. MA thesis at Ca' Foscari University of Venice. https://unitesi.unive.it/handle/20.500.14247/10291 [the commentary will be expanded and integrated in the digital scholarly edition as part of the Cimbrian Corpus project.
